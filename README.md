A customizable installer for building a master download station on Bananian for banana PI.

Currently planned are:

<ul>
  <li>CouchPotato</li>
</ul>

TV Shows:
<ul>

</ul>

Music:
<ul>
  <li>Headphones</li>
</ul>

eBooks:
<ul>

</ul>

<ul>

</ul>

  <li>nZEDb</li>
  <li>SABNzbd</li>
</ul>

</ul>
